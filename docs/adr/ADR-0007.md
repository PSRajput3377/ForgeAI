# ADR-0007 — uv for Python, pinned to 3.12

- **Status:** Accepted
- **Phase:** 1

## Context
We need fast, reproducible Python dependency management. The dev host runs
Python 3.14; the project targets 3.12.

## Decision
Use `uv` (not Poetry/pip) with a committed `uv.lock`. Pin Python to 3.12 in the
API Dockerfile and `pyproject.toml`; Docker is the source of truth.

## Consequences
- ✅ Much faster installs; reproducible lockfile.
- ✅ Build environment independent of the host Python.
- **Rejected:** Poetry (not installed, no advantage here); bare pip (no lock).
