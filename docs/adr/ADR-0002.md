# ADR-0002 — The Engineering Manager never writes code

- **Status:** Accepted
- **Phase:** 0

## Context
If the orchestrator also writes code, responsibilities blur and the system
becomes hard to reason about and test.

## Decision
The Manager only interprets the request, decomposes it, delegates, sequences,
monitors, handles failures, and produces the final response. All code is
produced by specialist agents.

## Consequences
- ✅ Clean orchestration layer; each specialist stays focused.
- ✅ Mirrors a real tech lead — intuitive mental model.
- ✅ Enforced and tested (`test_manager_never_writes_code`).
- ➖ An extra hop for trivial requests, accepted for consistency.
