# ADR-0001 — Build as a multi-agent team, not a single LLM call

- **Status:** Accepted
- **Phase:** 0

## Context
Single-model coding tools do everything in one call: understand, plan, write,
verify. There is no division of labor, no checkpoint, no second opinion.

## Decision
ForgeAI orchestrates specialist agents (Planner, Researcher, Coder, Tester,
Reviewer, DevOps, …) under an Engineering Manager.

## Consequences
- ✅ Separation of concerns; tighter per-agent prompts.
- ✅ Checkpoints — Tester/Reviewer can reject bad work before it ships.
- ✅ Traceability — every step is observable.
- ➖ More moving parts and orchestration complexity (addressed by ADR-0010).
- **Rejected:** single do-everything chatbot.
