# ADR-0005 — Ruthless MVP scope for v1.0

- **Status:** Accepted
- **Phase:** 0

## Context
Projects fail by trying to build everything at once.

## Decision
v1.0 = Login, Create Project, Chat, Planner + Coder + Review agents, local file
editing, Docker execution, project memory. Slack, Jira, AWS deployment, and
browser automation are explicitly deferred.

## Consequences
- ✅ Forces focus on the core team-of-agents loop.
- ✅ A clear "is this in scope?" test for every proposed feature.
- ➖ Some desirable features wait — intentional.
