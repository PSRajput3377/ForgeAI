# ADR-0010 — LangGraph over one shared ProjectState

- **Status:** Accepted
- **Phase:** 2

## Context
The multi-agent system (ADR-0001) needs an orchestration mechanism that makes
execution order, branching, and retries explicit and inspectable.

## Decision
Orchestrate with an explicit LangGraph `StateGraph` over a single
`ProjectState`. A conditional edge after review drives a bounded
`reflection → coder` retry loop. No agent calls another directly.

## Consequences
- ✅ Order/branching/retries are explicit and observable.
- ✅ Loose coupling — agents only touch shared state (ADR-0005 of the specs).
- ✅ Supports self-correction (reflection).
- ➖ A dependency on LangGraph's state model.
- **Rejected:** ad-hoc async function chains (unobservable, hard to retry).
