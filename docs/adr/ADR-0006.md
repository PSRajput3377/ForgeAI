# ADR-0006 — Monorepo with apps/ + packages/

- **Status:** Accepted
- **Phase:** 1

## Context
ForgeAI will grow to 10+ agents, 30+ APIs, 100+ prompts, and multiple LLMs. A
flat `frontend/` + `backend/` split would not scale.

## Decision
A single repository split into `apps/` (web, api) and `packages/` (agents,
core, prompts, tools, memory, rag, models), plus `infrastructure/`, `docs/`,
`specs/`, `scripts/`.

## Consequences
- ✅ Each concern isolated and independently evolvable.
- ✅ One toolchain, one Docker Compose, atomic cross-cutting commits.
- ➖ Requires `PYTHONPATH`/workspace wiring (see ADR-0009).
