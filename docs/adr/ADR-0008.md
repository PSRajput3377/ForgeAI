# ADR-0008 — Hand-scaffold the app starters

- **Status:** Accepted
- **Phase:** 1

## Context
`create-next-app` and similar generators are interactive, pull large trees, and
produce layouts we'd immediately rework.

## Decision
Write minimal Next.js 15 and FastAPI starters by hand; let dependencies resolve
at install/build time.

## Consequences
- ✅ Deterministic, fully version-controlled layout; no interactive prompts.
- ✅ Verified working: backend tests + lint pass, frontend type-checks + builds.
- ➖ We maintain the boilerplate ourselves (small, one-time cost).
