# ADR-0011 — EchoProvider for offline, deterministic testing

- **Status:** Accepted
- **Phase:** 2

## Context
The whole agent graph must be testable in CI without a live Ollama server or
the ~20 GB of pulled models.

## Decision
Ship an `EchoProvider` implementing `LLMProvider` that returns canned,
deterministic responses with no network calls. The full workflow test suite
runs against it.

## Consequences
- ✅ Fast, hermetic tests; the entire graph (incl. reflection loop) runs offline.
- ✅ Made trivial by the Model Router (ADR-0003).
- ➖ Echo output is not semantically meaningful — tests assert plumbing, not LLM
  quality (model-quality evaluation is covered by `specs/evaluation-spec.md`).
